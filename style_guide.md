## Principles

Readable before decorative.

Consistent spacing.

Minimal visual clutter.

Strategic information should always be prioritised.

Accuracy before appearance remains the first principle.

One logical milestone should be delivered per iteration.

---

## Sprite Rules

Approved Sprite Pack v3.

Do not resize individual sprite types independently.

Maintain consistent visual scale.

---

## Command Centre Direction

Command Centre is the Season 1 home/front-page workspace.

It should prioritise:

- strategic readability
- low clutter
- fast workspace switching
- consistent spacing and hierarchy

---

## Server Dock Placement

The server dock should sit bottom-left, just outside the map viewport.

Dock requirements:

- include Command Centre entry
- include eight Season 1 server entries
- show active workspace/server clearly
- never cover the playable map area

---

## Strategic Summary Presentation

Summary blocks should be glanceable and consistent.

Required summary set:

- Ice Crystals
- tiles owned
- territory percentage
- captured vs available structures
